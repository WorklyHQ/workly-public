## üé® Logo

### Primary Logo
- **Filename:** `workly_logo.png`
- **Format:** PNG with transparency
- **Minimum size:** 200x200px
- **Usage:** Main logo for all materials

- **Filename:** `workly_icon.ico`
- **Format:** ICO (multi-resolution)

- **Light background:** Dark logo

- **Monochrome:** Black/white versions for printing

## üî§ Typography

### Primary Font
- **Font:** Inter, Segoe UI, or system default
- **Weights:** Regular (400), Medium (500), Bold (700)
- **Usage:** UI, documentation, web

### Headings
- **Size:** 32px - 48px
- **Weight:** Bold (700)
- **Line height:** 1.2

### Body Text
- **Size:** 16px - 18px
- **Weight:** Regular (400)
- **Line height:** 1.6

### Code/Monospace
- **Font:** JetBrains Mono, Consolas, or system monospace
- **Size:** 14px - 16px
- **Usage:** Code blocks, technical content

## üìê Spacing & Layout

### Spacing Scale
- **XS:** 4px
- **S:** 8px
- **M:** 16px
- **L:** 24px
- **XL:** 32px
- **XXL:** 48px

### Grid System
- **Container:** 1200px max-width
- **Columns:** 12-column grid
- **Gutters:** 24px

### Borders
- **Radius:** 8px (standard), 16px (large cards)
- **Width:** 1px (subtle), 2px (prominent)
- **Color:** Medium gray or brand color
