## üé® Screenshot Guidelines

When adding screenshots:

4. **Privacy:** Remove any personal information
5. **Consistency:** Use the same theme/avatar across screenshots
6. **Lighting:** Ensure good contrast and visibility
7. **Cropping:** Remove unnecessary empty space
