## üé• Recording Guidelines

### Software Recommendations

**Windows:**
- [OBS Studio](https://obsproject.com/) - Free, professional recording
- [ShareX](https://getsharex.com/) - Quick clips and GIFs
- [Windows Game Bar](https://support.xbox.com/windows/xbox-game-bar) - Built-in (Win+G)

**Settings:**
- **Resolution:** 1920x1080 (1080p minimum)
- **Frame rate:** 60 FPS for smooth animations
- **Bitrate:** 8000-12000 kbps for high quality
- **Format:** MP4 (H.264 codec)

### Recording Tips

1. **Preparation:**
   - Clean desktop background
   - Close unnecessary applications
   - Disable notifications
   - Use consistent avatar/theme

2. **Content:**
   - Show real functionality
   - Keep it concise (1-5 minutes)
   - Focus on one feature per video
   - Include clear audio (if narrating)

3. **Quality:**
   - Good lighting (if showing setup)
   - Smooth mouse movements
   - Clear, visible UI elements
   - No personal information visible

4. **Editing:**
   - Trim dead space
   - Add captions if helpful
   - Include intro/outro (optional)
   - Add background music (optional, low volume)
